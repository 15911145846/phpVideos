# phpVideos
php 写的视频下载工具，现支持：Youku、Miaopai

# 环境依赖
*   PHP >= 7.1.3
*   OpenSSL PHP Extension
*   FFmpeg：
    *   Mac：brew install ffmpeg
    *   Linux:  [Download](http://ffmpeg.org/download.html)
*   cURL  >= 7.37.0
    
#   使用
php start 'link_address'

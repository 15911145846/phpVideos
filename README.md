# phpVideos
php 写的视频下载工具，现支持：Youku、Miaopai、腾讯、XVideos、Pornhub、91（不提供地址）

# 环境依赖
*   PHP >= 7.1.3
*   OpenSSL PHP Extension
*   cURL PHP Extension
*   FFmpeg：
    *   Mac：brew install ffmpeg
    *   Linux:  [Download](http://ffmpeg.org/download.html)
*   cURL  >= 7.37.0
